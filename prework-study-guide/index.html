HTML

  • The head element contains information about a webpage.
  • The body element represents the visible content shown to the user.

CSS

  • margin: defines the space around the outside an element
  • padding: defines the space around the content inside an element

Git

  • git status: check what branch currently on and show modified files in working directory
  • git checkout: switch to another branch and check it out into working directory
  • git checkout -b "branch-name": create and switch to new branch
  • git add -A: commit ALL (as flagged by '-A') files in the working branch
